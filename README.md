# sscom
Linux Version Serial Port Debugging Assistant

Forked from https://github.com/kent0628/sscom

Translated by ChatGPT 4o / Johan

How To build
```
sudo apt-get install qtcreator libqt5serialport5-dev libudev-dev

qmake 

make
```
